# Component prop value

Prop value used in component Slider

## Value
value must suit to the props range

## Usage
```jsx
import { Slider, Window } from 'lvlgjs-ui'
import { useState } from 'react'

function Component () {
    const [value, setValue] = useState()
    return (
        <Window>
            {/* controlled */}
            <Slider
              style={style.slider}
              onChange={(e) => setValue(e.value)}
              value={value}
              range={[0, 50]}
            />
            {/* not controlled */}
            <Slider
              style={style.slider}
              onChange={(e) => console.log(e)}
              range={[0, 100]}
            />
        </Window>
    )
}

const style = {
    slider: {},
}
```
